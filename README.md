## Run the game
stack run in the terminal

## Controls in game
Forward thrust                               = W  
Steering right                               = A  
Steering left                                = D  
Shooting                                     = Space  
Pause / Unpause                              = Tab  
Restart (Only on dead screen)                = R
Return to level select (Only on dead screen) = Enter

## Controls in level select
Level down                                   = S       
Level up                                     = W  
Select level                                 = Enter
